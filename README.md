## 🌌 NebulaTouch

**Gesture-Controlled 3D Particle System**

NebulaTouch is an interactive 3D particle system where users can manipulate and morph particle structures in real time using hand gestures captured through the webcam.

This project explores the intersection of **computer vision, real-time rendering, and AI-assisted development**, creating a touchless, immersive visual experience.

---

## ✨ Features

* 🖐️ **Real-time hand tracking** via webcam
* 🌌 **Dynamic particle system** built with Three.js
* 🔄 **Shape morphing templates**:

  * Saturn
  * Hearts
  * Flowers
  * Buddha silhouette
  * Fireworks
* 🎨 **Live particle color control** (gesture + UI)
* ⚡ **Smooth animations** using requestAnimationFrame
* 🧠 **Gesture-based interaction** using hand tension detection
* 🖥️ **Minimal modern UI** for seamless control

---

## 🛠️ Tech Stack

* **Three.js** (3D rendering)
* **JavaScript (Vanilla)**
* **MediaPipe / TensorFlow.js** (hand tracking)
* **HTML + CSS**

---

## 🧪 How It Works

The system tracks hand landmarks in real time using a browser-based hand tracking model.

A custom **“tension metric”** is calculated based on the distance between fingertips and the palm.

This value controls:

* Particle system scaling
* Expansion and contraction behavior

Particles dynamically reposition themselves when switching between templates, creating smooth morphing transitions between shapes.
